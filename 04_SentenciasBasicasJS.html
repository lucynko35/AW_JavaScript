Sentencias básicas en JS

Las líneas de código terminan siempre con punto y coma (;)

Es recomendable utilizar ; al final de cada sentencia

Se puede escribir varias sentencias en una misma línea separándolas con ;

Espacios en blanco

Los múltiples espacios en blanco se ignoran

Se recomienda utilizar espacios entre operadores =, +, -, /

Líneas de más de 80 caracteres

Es recomendable dividir la linea después de un operador

Bloque de código

Se agrupan entre llaves {}

Se utilizan en funciones, bucles, condiciones, etc

Palabras reservadas

  • break
  • continue
  • switch
  • for
  • do ...while
  • if ...else
  • try ..catch
  • var
  • function
  • return
